Definición

Pishing

El phishing es una técnica que utiliza mensajes falsos o engañosos para obtener información personal o confidencial de las personas, como contraseñas o números de tarjetas de crédito. Ejemplo: Un correo electrónico que aparenta ser de tu banco y te pide que ingreses tu nombre de usuario y contraseña en un sitio web falso, con el fin de robar tu información bancaria.

Definición

Ataque de denegación de servicio (DDoS)

En un escenario de DDoS, un atacante utiliza una red de dispositivos comprometidos, conocidos como "bots" o "zombies", para enviar una gran cantidad de tráfico o solicitudes al objetivo de manera simultánea. Estos bots pueden ser computadoras infectadas previamente por malware o dispositivos de Internet de las Cosas (IoT) comprometidos. El objetivo principal del ataque es saturar los recursos del sistema objetivo, como el ancho de banda de la red, la capacidad de procesamiento o la memoria, lo que resulta en una interrupción o degradación del servicio para los usuarios legítimos.

Definición

Ataque de Fuerza Bruta

En un ataque de fuerza bruta, un atacante intenta descubrir una contraseña o clave secreta probando todas las posibles combinaciones hasta encontrar la correcta. Este tipo de ataque se basa en la suposición de que, con suficiente tiempo y recursos computacionales, es posible encontrar la contraseña correcta. Ejemplo: Un atacante utiliza un programa automatizado para probar todas las combinaciones posibles de contraseñas en una cuenta de usuario en línea hasta que se encuentra la contraseña correcta y se obtiene acceso no autorizado.

Definición

Ataque de Inyeccion de Código

En un ataque de inyección de código, un atacante inserta código malicioso en una aplicación o sistema vulnerable para ejecutar comandos no autorizados. El ataque más común es la inyección de SQL, donde se aprovechan las vulnerabilidades en las consultas SQL de una aplicación web para manipular la base de datos subyacente y obtener acceso no autorizado a datos sensibles. Ejemplo: Un atacante introduce código SQL malicioso en un campo de entrada de un formulario web para obtener acceso no autorizado a la base de datos subyacente.
